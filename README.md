# Free_API
A restaurant menu API with allergens.


## Description
This is a restaurant menu API. It's possible to add, edit and delete a dish, see and edit other informations, but only if logged in. To do so, it's necessary to sign up.

## Built with
* C#
* .Net 6
* Swagger
* SQL database
